**lambda表达式**

```python
name = lambda [list] : 表达式
```

又称匿名函数，[list] 作为可选参数，等同于定义函数是指定的参数列表；value 为该表达式的名称。

```python
# x + y
add = lambda x,y:x+y
print(add(3,4))
```
